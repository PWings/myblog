JVM的架构模型

JVM编译器输入的指令流基本上是基于栈的指令架构，还有一种是基于寄存器的指令架构

这两种架构的区别

  • 栈 与硬件的耦合度底，移植性强
  • 寄存器 相当与某种硬件的定制版，特定硬件上面性能高，移植性差
